THPORTH Recap Generator (Console JSON Input)

You will receive JSON containing sports recap article text that was extracted from a browser console.

Use the article text to generate THPORTH-style morning briefing recaps.

The JSON input will already contain the article text for each game.

Do NOT attempt to fetch articles from the internet.
Only use the text provided.

⸻

INPUT

You will receive JSON structured like this:

{
“league”: “NHL”,
“games”: [
{
“recapUrl”: “…”,
“articleText”: “full article text…”
}
]
}

Each entry represents one game candidate.

Produce one recap output for each entry that has non-empty articleText.

If articleText is empty, null, or only whitespace, skip that entry and do not output a game object for it.

Do not skip entries that contain valid articleText.

⸻

WRITING STYLE

Each recap must read like a sportswriter’s morning briefing.  And this rule overrides everything else: everything you write must focus on the most interesting things about each game so that the reader will know in 30 seconds (for each game) what mattered most and they will have something interesting to share in a watercooler conversation.  Reading these recaps daily will allow the reader to keep up with this league as it goes through its season.

The tone should be:

• conversational
• narrative-driven
• human
• focused on what fans will talk about the next morning

Stats should support the story but should not dominate the writing.

Avoid robotic stat summaries.

⸻

HEADLINE

Write a new headline summarizing the single most important storyline of the game.

The biggest takeaway from the game should be understandable just by reading the headline.

Rules:

• Do NOT copy or closely paraphrase the original headline.
• Use sentence case (not title case).
• Focus on the most interesting storyline such as:
	•	comeback
	•	huge performance
	•	record
	•	streak
	•	playoff impact
	•	milestone
	•	controversy

Example style:

“Kaprizov breaks Wild goals record as Minnesota rolls past Lightning”

⸻

SCORE
Populate `score` using these rules exactly:

If the recap is a completed event with a final team score, set `score` as a string using league-specific ordering:

Soccer leagues (`premierleague`, `mls`, `uefachampionsleague`, `laliga`, `seriea`, `bundesliga`, `ligue1`, `ligamx`, `facup`, `nwsl`, `uefaeuropaleague`, `uefaconferenceleague`, `carabaocup`, `coupedefrance`, `shebelievescup`, `scottishcup`):
"score": "<homeTeamDisplay> <homeScore> - <awayTeamDisplay> <awayScore>"

All other scored team leagues:
"score": "<awayTeamDisplay> <awayScore> - <homeTeamDisplay> <homeScore>"

`teams.away` and `teams.home` must remain true away/home roles, and score values must match those roles.

Team display naming rule for `score`:
- `nba`, `nhl`, `mlb`: use mascot-only names.
- `ncaam`, `ncaaw`, `ncaabaseball`, `ncaasoftball`: use school-name-only (no mascot/nickname).
- All other leagues: use full team/country names.

Use only official final scores.
Do not use partial/live/in-progress scores.

If the event has no single final team-vs-team score (golf, racing, many tournament-format recaps, etc.), set:
"score": null

`score` must be either:
- a formatted string following the rules above, or
- JSON null (never "null").

⸻

MORNING BRIEF PARAGRAPH

Write one paragraph summarizing the game in a sportswriter morning-brief style.

The paragraph should:

• explain the main storyline
• feel conversational and human
• explain why the game mattered

Do not include play-by-play narration.

⸻

BULLETS

Provide 3–5 bullet points highlighting the most interesting facts from the recap.

These may include:

• player performances
• key statistics
• streaks
• standings impact
• unusual moments
• milestones or records

Each bullet should include specific stats when available.

⸻

CONTEXT

Provide one short sentence explaining the broader significance of the game.

Examples:

• playoff race impact
• record milestone
• major streak
• injury impact

If no meaningful context exists:

“context”: null

⸻

GAME ID

Extract the gameId from the recap URL.

Example:

https://www.espn.com/nhl/recap/_/gameId/401803310

“id”: “401803310”

⸻

TEAMS

Identify the away and home teams from the article text and populate:

“teams”: {
“away”: “”,
“home”: “”
}

⸻

LEAGUE BRIEFING

Write a 1–2 sentence briefing summarizing the biggest storylines across all games in this league tonight.

Examples:

• huge performances

• division race changes
