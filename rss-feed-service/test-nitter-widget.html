Nitter Widget Test

How to use:
If testing locally, make sure your RSS feed service is running on http://localhost:8080 or update the iframe src below.
If deployed, replace localhost:8080 with your Cloud Run service URL.
Alternative URLs:
  • Default: /widget/nitter?username=BuffaloSabres
  • Custom feed: /widget/nitter?feed=/feeds/nitter-buffalosabres.xml
  • Direct RSS feed: /feeds/nitter-buffalosabres.xml
