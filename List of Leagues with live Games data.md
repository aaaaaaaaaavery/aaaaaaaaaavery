## Summary

**Total: 85+ leagues** across multiple sports categories, with the Featured section appearing first, followed by leagues in the order specified in the `LEAGUE_DISPLAY_ORDER` configuration.

### Data Source
All live game data is fetched from the FlashLive API via the `index.js` backend script, filtered through the `ALLOWED_LEAGUE_KEYWORDS` configuration, and displayed on the home page Games column in real-time.
