NCAA Baseball JSON Merge + Sort Instructions

Use this whenever I paste multiple JSON blocks into `recaps-manual/daily/ncaabaseball.json`.

Rules
1. Merge all pasted JSON blocks into one valid JSON object/file.
2. Fix all syntax issues (commas, brackets, braces, quotes, array/object nesting).
3. Keep every valid game entry unless it is a true duplicate.
4. Normalize each game's `highlights` to the required structure:
   "highlights": [
     {
       "provider": "youtube",
       "videoId": "",
       "title": "",
       "url": "https://www.youtube.com/watch?v=...",
       "embedUrl": "https://www.youtube.com/embed/..."
     }
   ]
5. Derive `videoId` from the YouTube URL when needed.
6. Reorder games by team-priority mapping below.

Sorting logic
- Sort games by the first team in this priority list that appears in either `teams.away` or `teams.home`.
- Earlier team in the list means higher priority (appears first in output).
- If both teams are in the list, use the team with the higher priority (closest to top).
- If no listed team matches, keep those games after all matched games (stable order).
