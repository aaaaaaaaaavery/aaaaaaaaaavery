## Feed Replacement Guide

### How to Replace Feeds in index.html

1. Find the old RSS.app feed URL (e.g., `https://rss.app/feeds/pcL7F0hcd8SbqKlP.xml`)
2. Replace it with the new feed URL (e.g., `https://rss-feed-service-124291936014.us-central1.run.app/feeds/youtube-tennischannel.xml`)
3. Or use the local URL if running locally: `http://localhost:8080/feeds/youtube-tennischannel.xml`
